# 第6章 ジェネレータ

遅延評価による無限列、`yield from` による委譲、`send()` によるコルーチンを扱う問題です。

## 問題一覧

- [0601: 無限素数ストリーム](0601-prime-stream.md) — 上限を決めない増分エラトステネス篩で、素数を無限に生成するジェネレータを実装します。
- [0602: same fringe 問題](0602-same-fringe.md) — 入れ子タプルの木を遅延走査し、2つの木の葉の列が等しいかを判定します。
- [0603: オンライン分散のコルーチン](0603-online-variance.md) — send() で値を受け取るたびに平均と分散を Welford 法で更新する拡張ジェネレータを実装します。
- [0604: ハミング数のマージ](0604-hamming-numbers.md) — 素因数が 2, 3, 5 に限られる数を昇順かつ重複なしで無限に生成します。
- [0605: ラウンドロビン・スケジューラ](0605-round-robin-scheduler.md) — ジェネレータをタスクとして登録し deque でラウンドロビン実行する協調的スケジューラを実装します。
