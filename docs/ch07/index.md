# 第7章 クラスとオブジェクト指向プログラミング

MRO、ディスクリプタ、メタクラス、弱参照といったオブジェクトシステムの深部を扱う問題です。

## 問題一覧

- [0701: C3線型化の実装](0701-c3-linearization.md) — クラス名の継承グラフからメソッド解決順序を計算する C3 線型化アルゴリズムを実装します。
- [0702: ディスクリプタで作る単位付き量](0702-unit-descriptor.md) — 属性への代入時に物理量の次元を検査するデータディスクリプタを実装します。
- [0703: メタクラスによるハッシュコンシング](0703-hash-consing.md) — メタクラスの `__call__` をメモ化し、同じ引数からは同一のインスタンスを返すようにします。
- [0704: 弱参照キャッシュ](0704-weakref-cache.md) — 強参照が消えるとエントリが自動的に消えるノードキャッシュを弱参照辞書で実装します。
- [0705: 代数構造の抽象基底クラス](0705-monoid-abc.md) — モノイドの抽象基底クラスと、繰り返し二乗法による冪演算ミックスインを実装します。
- [0706: \_\_slots\_\_ で作る Union-Find](0706-slots-union-find.md) — `__slots__` を持つクラスとして素集合データ構造を実装します。
