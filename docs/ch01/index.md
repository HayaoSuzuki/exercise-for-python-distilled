# 第1章 Pythonの基礎

「第1章 Pythonの基礎」を終えた人のための総合問題です。データ構造の使い分けと計算量を意識する構成です。

## 問題一覧

- [0101: 循環小数の循環節](0101-repeating-decimal.md) — 筆算の除法を模した整数演算で、分数の十進小数展開に現れる循環節を求めます。
- [0102: 転倒数](0102-inversion-count.md) — 列の乱れ具合を測る転倒数を、分割統治で O(n log n) 時間で数えます。
- [0103: スタック2つで作るキュー](0103-two-stack-queue.md) — append と pop しかできないスタック2本から、償却 O(1) のキューを組み立てます。
- [0104: テキストのエントロピー](0104-text-entropy.md) — 文字の出現頻度からシャノンエントロピーを計算します。
- [0105: ニュートン法による整数平方根](0105-integer-sqrt.md) — 整数演算だけのニュートン法で平方根の整数部分を求めます。
