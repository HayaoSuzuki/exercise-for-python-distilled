# 第10章 組み込み関数と標準ライブラリ

組み込み関数だけを道具に、数論・線形代数・ソート理論のアルゴリズムを組み立てる問題です。

## 問題一覧

- [1001: 例外階層の最近共通祖先](1001-exception-lca.md) — except 節のマッチング規則を根拠に、複数の例外クラスをまとめて捕捉できる最も特殊な基底クラスを `__mro__` から求めます。
- [1002: 3引数 pow による Miller–Rabin](1002-miller-rabin.md) — 3引数の pow と divmod だけを道具に、Miller–Rabin の確率的素数判定を決定的な底の集合で実装します。
- [1003: 安定ソートで作る基数ソート](1003-radix-sort.md) — sorted の安定性を桁ごとに積み重ねて、非負整数の LSD 基数ソートを実装します。
- [1004: zip と map の線形代数](1004-builtin-linear-algebra.md) — タプルのタプルで表した行列の転置・積・累乗を zip、map、sum で組み立て、Fibonacci 数を対数時間で求めます。
- [1005: \_\_format\_\_ とローマ数字](1005-roman-format.md) — 整数を包むクラスに `__format__` を実装し、f文字列から使える独自の書式ミニ言語でローマ数字表記を提供します。
