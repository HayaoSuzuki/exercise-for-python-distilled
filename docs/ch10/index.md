# 第10章 組み込み関数と標準ライブラリ

基本的な組み込み関数による集計を確認してから、数論・線形代数・ソート理論のアルゴリズムに進みます。

## 問題一覧

- [1000: 数値列の要約](1000-number-summary.md)：組み込み関数を使い、数値列の個数、合計、最小値、最大値を求めます。
- [1001: 上位の点数](1001-top-scores.md)：組み込み関数 sorted を使い、点数の高い項目を取り出します。
- [1050: 例外階層の最近共通祖先](1050-exception-lca.md)：except 節のマッチング規則を根拠に、複数の例外クラスをまとめて捕捉できる最も特殊な基底クラスを __mro__ から求めます。
- [1051: 3引数 pow による Miller–Rabin](1051-miller-rabin.md)：3引数の pow と divmod だけを道具に、Miller–Rabin の確率的素数判定を決定的な底の集合で実装します。
- [1052: 安定ソートで作る基数ソート](1052-radix-sort.md)：sorted の安定性を桁ごとに積み重ねて、非負整数の LSD 基数ソートを実装します。
- [1053: zip と map の線形代数](1053-builtin-linear-algebra.md)：タプルのタプルで表した行列に対する転置・積・累乗を zip、map、sum で組み立て、Fibonacci 数を対数時間で求めます。
- [1054: __format__ とローマ数字](1054-roman-format.md)：整数を包むクラスに __format__ を実装し、f文字列から使える独自の書式ミニ言語でローマ数字表記を提供します。
