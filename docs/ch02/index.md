# 第2章 演算子、式、データ操作

内包表記、集合演算、シーケンス操作を数学の道具として使う問題です。

## 問題一覧

- [0201: 内包表記で作る関係代数](0201-relational-algebra.md) — 辞書のリストで表した関係に対する選択・射影・自然結合を内包表記で実装します。
- [0202: 集合演算による推移閉包](0202-transitive-closure.md) — 有向グラフの辺集合の推移閉包を集合演算の不動点反復で求めます。
- [0203: 辞書で表す疎多項式](0203-sparse-polynomial.md) — {次数: 係数} の辞書で表した疎多項式の加算・乗算・評価を実装します。
- [0204: スライスで作る完全シャッフル](0204-riffle-shuffle.md) — アウトシャッフルをスライスで実装し、デッキが元の並びに戻るまでの回数を求めます。
- [0205: ジェネレータ式と包除原理](0205-inclusion-exclusion.md) — 与えられた素数のどれとも互いに素な整数の個数をジェネレータ式で数えます。
