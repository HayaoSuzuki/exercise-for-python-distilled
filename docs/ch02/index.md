# 第2章 演算子、式、データ操作

辞書、集合、内包表記によるデータ操作を確認してから、関係、閉包、多項式などの題材に進みます。

## 問題一覧

- [0200: 在庫差分](0200-inventory-delta.md)：2つの辞書を比べ、品目ごとの増減を計算します。
- [0201: タプルの左回転](0201-rotate-left.md)：スライスと剰余演算を使い、タプルの要素を左へ回転します。
- [0202: 偶数だけの平方](0202-even-squares.md)：条件付き内包表記で、偶数だけを選んで平方のリストを作ります。
- [0250: 内包表記で作る関係代数](0250-relational-algebra.md)：辞書のリストで表した関係に対する選択・射影・自然結合を内包表記で実装します。
- [0251: 集合演算による推移閉包](0251-transitive-closure.md)：有向グラフの辺集合の推移閉包を集合演算の不動点反復で求めます。
- [0252: 辞書で表す疎多項式](0252-sparse-polynomial.md)：{次数: 係数} の辞書で表した疎多項式の加算・乗算・評価を実装します。
- [0253: スライスで作る完全シャッフル](0253-riffle-shuffle.md)：アウトシャッフルをスライスで実装し、デッキが元の並びに戻るまでの回数を求めます。
- [0254: ジェネレータ式と包除原理](0254-inclusion-exclusion.md)：与えられた素数のどれとも互いに素な整数の個数をジェネレータ式で数えます。
