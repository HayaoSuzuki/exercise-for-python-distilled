# 第4章 オブジェクト、型、プロトコル

特殊メソッドのプロトコルを実装して、数学的な対象を Python のオブジェクトとしてモデル化する問題です。

## 問題一覧

- [0401: 剰余環 Z/nZ](0401-modular-ring.md) — 法 n の合同算術を数値プロトコルで実装し、`__eq__` と `__hash__` の整合性を保ちます。
- [0402: 半順序としての多重集合包含](0402-multiset-order.md) — 多重集合の包含関係を比較プロトコルで定義し、比較不能な対を観察します。
- [0403: 不変連結リスト](0403-cons-list.md) — 構造共有するイミュータブルな連結リストをコンテナプロトコルとイテレータプロトコルで実装します。
- [0404: 遅延属性](0404-lazy-attribute.md) — `__getattr__` で初回アクセス時にだけ計算し、結果をインスタンス辞書に埋める遅延属性を実装します。
- [0405: 四元数](0405-quaternion.md) — 非可換な積を持つ四元数を数値プロトコルで実装します。
