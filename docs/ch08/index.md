# 第8章 モジュールとパッケージ

import システムの意味論をグラフ理論と形式的規則の問題として再実装する問題です。

## 問題一覧

- [0801: ミニ import システムと循環検出](0801-mini-import-system.md) — sys.modules 相当のキャッシュを持つミニ import システムを実装し、循環インポートを検出します。
- [0802: モジュール検索パスの解決](0802-module-search-path.md) — sys.path 相当の検索パスを順に探索する first-match のモジュール解決を実装します。
- [0803: 相対インポートの解決](0803-relative-import.md) — 相対インポートを PEP 328 の規則に従って絶対モジュール名へ変換します。
- [0804: リロード順序とトポロジカルソート](0804-reload-order.md) — モジュールの依存グラフから「依存されるものが先」のリロード順序を決定的に求めます。
- [0805: from module import * の意味論](0805-star-import.md) — `from module import *` が束縛する名前の集合を `__all__` とアンダースコア規則から求めます。
