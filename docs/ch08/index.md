# 第8章 モジュールとパッケージ

モジュールの公開名を確認してから、検索パス、相対インポート、リロード、スターインポートの意味論に進みます。

## 問題一覧

- [0800: モジュールの公開名](0800-exported-names.md)：名前空間から、外部に公開する名前の一覧を求めます。
- [0801: モジュール名からファイル名を作る](0801-module-filename.md)：ドット区切りのモジュール名を、相対的な Python ファイル名へ変換します。
- [0850: ミニ import システムと循環検出](0850-mini-import-system.md)：sys.modules 相当のキャッシュを持つミニ import システムを実装し、循環インポートを検出します。
- [0851: モジュール検索パスの解決](0851-module-search-path.md)：sys.path 相当の検索パスを順に探索する first-match のモジュール解決を実装します。
- [0852: 相対インポートの解決](0852-relative-import.md)：相対インポートを PEP 328 の規則に従って絶対モジュール名へ変換します。
- [0853: リロード順序とトポロジカルソート](0853-reload-order.md)：モジュールの依存グラフから「依存されるものが先」のリロード順序を決定的に求めます。
- [0854: from module import * の意味論](0854-star-import.md)：from module import * が束縛する名前の集合を __all__ とアンダースコア規則から求めます。
