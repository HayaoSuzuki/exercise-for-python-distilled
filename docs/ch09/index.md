# 第9章 入力と出力

文字符号化、バイナリ形式、ハッシュ、正規表現、非同期 I/O と、データ表現と並行処理の理論を扱う問題です。

## 問題一覧

- [0901: UTF-8 コーデックの手書き実装](0901-utf8-codec.md) — 可変長符号化の規則をビット演算だけで実装し、不正なバイト列の検出まで行います。
- [0902: struct によるチャンク形式のパース](0902-chunk-parser.md) — タグと長さを持つチャンクの繰り返しからなるバイナリ形式を解析します。
- [0903: hashlib で作る Merkle 木](0903-merkle-tree.md) — SHA-256 でハッシュ木を構成し、認証パスによる包含証明と検証を実装します。
- [0904: re で作る字句解析器](0904-regex-lexer.md) — 名前付きグループを合成した正規表現で最長一致のトークン化を行います。
- [0905: asyncio による有界並行実行](0905-bounded-gather.md) — セマフォで同時実行数を制限しながらコルーチン群を並行実行します。
