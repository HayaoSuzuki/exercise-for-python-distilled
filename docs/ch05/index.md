# 第5章 関数

高階関数、クロージャ、デコレータを題材に、関数型プログラミングと λ計算の考え方を扱う問題です。

## 問題一覧

- [0501: 関数合成のモノイド](0501-function-composition.md) — 1引数関数の合成 `compose` と反復合成 `iterate` を実装し、関数合成がモノイドをなすことを確認します。
- [0502: メモ化デコレータと動的計画法](0502-memoize-dp.md) — `functools.wraps` を使ったメモ化デコレータを自作し、重複部分問題を持つ再帰計算を多項式時間で扱います。
- [0503: 不動点コンビネータ](0503-fixed-point-combinator.md) — λ計算の Z コンビネータに倣い、名前による自己参照なしで再帰を実現する `fix` を実装します。
- [0504: トランポリンと継続渡し形式](0504-trampoline-cps.md) — サンクを返す関数をループで駆動するトランポリンを実装し、再帰制限を超える深さの計算を実行します。
- [0505: シグネチャによる自動カリー化](0505-auto-curry.md) — `inspect.signature` で仮引数の個数を調べ、位置引数が揃うまで部分適用を返す `curry` を実装します。
